---
title: Graph Memory
description: 'Knowledge Graph of your memories'
---


Mem0 uses a graph database to store memories in a knowledge graph. This allows for more complex queries and relationships between memories.

## Add

To create an knowledge graph of your memories, you can use the `enable_graph` property in the `add()` method.

```python
from mem0 import MemoryClient

m = MemoryClient(api_key="xxx")

m.add(
    messages=[
        {"role": "user", "content": "I like to eat pizza"},
    ],
    user_id="sam",
    agent_id="agent_1",
    enable_graph=True,
)
``` 
This will create a knowledge graph of the memories and store it in the graph database. 


## Search

Use the `search()` method with the `enable_graph` property to get the knowledge graph of the memories.

<CodeGroup>
```python With Format
result = m.search(
    query="preferences of sam",
    user_id="sam",
    agent_id="agent_1",
    enable_graph=True,
    output_format="v1.1"
)
print(result)
```

```python Without Format
result = m.search(
    query="preferences of sam",
    user_id="sam",
    agent_id="agent_1",
    enable_graph=True,
)
print(result)
```
</CodeGroup>

This will return the nodes and relationships that are related to the query "preferences of sam".

## Get All

Use the `get_all()` method with the `enable_graph` property to get all the memories in the graph database.

<CodeGroup>
```python With Format
result = client.get_all(
    user_id="sam",
    agent_id="agent_1",
    enable_graph=True,
    output_format="v1.1"
)
print(result)
```

```python Without Format
result = client.get_all(
    user_id="sam",
    agent_id="agent_1",
    enable_graph=True,
)
print(result)
```
</CodeGroup>

This will return all the memories in the graph database.
